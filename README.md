## 本地调试步骤:

本地调试步骤:
1. 添加环境变量: `export SOURCE_DIR="xxx/src_dir"`
2. 添加环境变量: `export TASK_REQUEST="xxx/task_request.json"`
3. 按需修改`task_request.json`文件中各字段的内容
4. 命令行`cd`到项目根目录,执行命令:  `python3 src/main.py`
